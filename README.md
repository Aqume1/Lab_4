# Lab_4
В каждом из 2 файлов python содержаться функции считающие площадь и периметр 2 фигур:
- круг
- квадрат

## Area
- Circle: S = πR²
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Square: P = 4a

## Functions
- ### circle.py
  - ##### area(r)
    
    Принимает число r - радиус круга, возвращает число возведенное в квадрат и домноженное на пи - площадь круга
print(area(6))
    -> 113.04
- ##### perimeter(r)
    
    Принимает число r - радиус круга, возвращает число домноженное на два и на пи - периметр круга
print(perimeter(6))
    -> 37.68
- ### square.py
  - ##### area(a)
    
    Принимает число a - сторону квадрата, возвращает квадрат числа - площадь квадрата
print(area(5))
    -> 25
- ##### perimeter(a)
    
    Принимает число a - сторону квадрата, возвращает произведение стороны квадрата на 4 - перметр квадрата
print(perimeter(5))
    -> 20
